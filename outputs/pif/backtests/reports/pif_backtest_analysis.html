Sanity Check Read

  • The mechanical checks are now clean: arithmetic violations are zero across all five strategies. `P1` through `P4` remain fully invested, while `P5` deliberately lets gross exposure fall and cash build when disclosed sells outpace disclosed buys.
  • Two corrections matter most for the final `PIF` read. Split-adjusted marking removed false raw-price reverse-split gains, and same-day multi-period filing bundles are now traded as a single latest observable sleeve instead of as a union of stale periods.
  • `P1` is weak because it is extremely concentrated. It carries only `6.47` names on average, its average max weight is `59.7%`, and it periodically goes all-in on ugly cohorts, including a worst single forward return of `-91.9%`.
  • `P3` does find stronger simple average forward returns in `accumulation_like` names (10.5% vs 1.5% for `neutral`), but the tilted portfolio still loses more because it concentrates into names with deeper tail losses, including a worst accumulation-like forward return of `-91.9%`.
  • `P4` does not rescue the sleeve. The names it avoided under the likely-reduction rule actually averaged `10.2%` forward return, versus `3.9%` for the names it kept, so the exit-avoidance heuristic was directionally wrong in this sample.
  • `P5` remains conceptually important even though it is no longer the top absolute-return strategy. Allowing cash to build keeps it positive at 48.9%, which is more intuitive than forcing every disclosed sale back into the surviving names.
  • The strict alpha question is benchmark-relative. `P2` is now the strongest absolute strategy at 122.3%, but it still trails `SPY` by -25.9%. `P5` lags `SPY` by -50.4% with information ratio -0.12. The corrected read is therefore “positive absolute results, but still no alpha versus `SPY`.”
