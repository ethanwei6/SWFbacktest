# PIF 13F Raw Data

Put raw `PIF` EDGAR filing files here.

Recommended structure:

- one directory per accession number
- include:
  - `primary_doc.xml`
  - XML information table file

Example:

- `000101143824000215/primary_doc.xml`
- `000101143824000215/informationtable.xml`

Do not edit raw files in place.
